# The Big Screen Club

A web application for creating movie clubs, suggesting and voting on movies, and comparing ratings.

## Project Structure

This is a monorepo containing:
- `frontend/` - React + TypeScript frontend (Vite)
- `backend/` - Node.js + Express + TypeScript backend

## Setup

### Frontend
```bash
cd the-big-screen-club/frontend
npm install
npm run dev
```

### Backend
```bash
cd the-big-screen-club/backend
npm install
npm run dev
```

## Development

- Frontend runs on `http://localhost:5173` (Vite default)
- Backend runs on `http://localhost:3000` (Express default)
